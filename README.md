# Alura Store Analysis

This project analyzes the performance of four Alura Store branches to determine which store should be sold.

## Analysis
The project evaluates:

- Total revenue per store
- Best-selling categories
- Customer reviews
- Most sold products
- Average shipping cost

## Tools
- Python
- Pandas
- Matplotlib

## Conclusion
Based on the analysis of revenue, customer reviews, and shipping performance, a recommendation is made for which store Mr. Juan should sell.
